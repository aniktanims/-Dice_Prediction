## 📊 Dataset Format

| dice1 | dice2 | dice3 | total |
| ----- | ----- | ----- | ----- |
| 3     | 2     | 6     | 11    |

* **dice1**, **dice2**, **dice3** → values of each die (1–6)
* **total** → optional; auto-corrected if mismatched
